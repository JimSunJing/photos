# 使用说明

## 添加图片

打开 `src/data/photos.ts`，在 `photos` 数组末尾添加新条目：

```ts
{
  id: "26",                        // 唯一编号，跟前面的接上
  src: "https://你的图片链接",       // 图片URL
  alt: "图片描述文字",               // 鼠标悬停/大图时显示
  category: "脂包肌叉腰20cm",        // 所属分类（见下方分类列表）
},
```

### 图片URL获取

图片建议上传到图床（如 ufs.sh、sm.ms、imgur 等），拿到直链复制到 `src` 字段。

### 添加多张

直接在数组最后加逗号继续写：

```ts
{
  id: "26",
  src: "https://...",
  alt: "新图片1",
  category: "配件",
},
{
  id: "27",
  src: "https://...",
  alt: "新图片2",
  category: "配件",
},
```

---

## 新增分类

在 `src/data/photos.ts` 中找到 `categories` 数组，添加新分类名：

```ts
export const categories = [
  "全部",
  "脂包肌叉腰20cm",
  "肌肉展示中20cm",
  "脂包肌稍息12cm",
  "脂包肌稍息25cm",
  "配件",
  "其他",
  "新分类名",   // ← 在这里加
] as const;
```

> ⚠️ `"全部"` 必须保留，不要删掉。

然后在图片的 `category` 字段里填上对应的分类名即可。

---

## 修改/删除图片

找到对应 id 的条目，直接改字段或删掉整段 `{ ... },`。

---

## 推送到线上

改完之后在项目目录打开终端，执行：

```bash
git add src/data/photos.ts
git commit -m "更新图片"
git push
```

Vercel 会自动部署，约 1-2 分钟生效。
